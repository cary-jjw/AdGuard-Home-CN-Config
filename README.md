1. 配置前准备工作
由于网络配置环境有别 ，建议在配置前（非部署AdGuard Home环境）利用  https://www.dnsip.cn/ 进行DNS测速。
建议多测试几次按均值记录。另外 https://www.dnsip.cn/ 测试是是基于IPV4 DNS 如果是要配置DOT DOH 就没有关系了
